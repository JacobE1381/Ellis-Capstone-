README

Download Ellis_Capstone_Project.ipynb
Download AIM_240_Dataset.npz

You may need to install these onto your computer
pip install torch==2.0.1
pip install keras==2.13.1
pip install numpy==1.26.0
pip install tqdm==4.66.1
pip install matplotlib==3.8.0

Once downloaded you can upload the .ipynb to your google drive and open with google colab
Now you can upload the AIM_240_Dataset.npz file into the Files section of the google colab
At this point you should be able to run all the google colab and view the model output at the end of the program
